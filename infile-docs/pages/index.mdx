# Bienvenido a Infile Docs

Documentación técnica oficial de los **Web Services de INFILE** para la integración con el régimen de Facturación Electrónica en Línea (**FEL**) de Guatemala.

## ¿Qué encontrarás aquí?

Esta documentación cubre los servicios disponibles para desarrolladores e integradores que desean conectar sus sistemas ERP, aplicaciones o plataformas con los servicios de INFILE.

## Requisitos generales

Para consumir cualquiera de los servicios necesitas:

- **Credenciales activas** provistas por INFILE (Usuario API, Llave API)
- Conocimiento básico en consumo de servicios **REST**
- Herramienta de pruebas como [Postman](https://www.postman.com/) o similar
